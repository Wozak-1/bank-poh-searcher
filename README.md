# WHERE’S MY STUFF?

A plugin that takes snapshots of your bank and POH storage locations in order to make searching for fashionscape easier!
I plan on adding STASH's soon

## Features
- Snapshots bank and POH storage when opened
- Search across all stored items
- Shows quantity and GE value
- Sorting functionality by name, value, recently changed. Particularly useful for dumping loot and comparing gains/losses

## Notes
- Open a storage interface to update it. POH currently requires each one opened to be stored.
- Uses GE prices when available otherwise uses HA value
